### 03. Решение линейного уравнения

```Python
print("Я умею решать линейные уравнения вида a * x + b = c.")
a = float(input("Введите коэффициент a: "))
b = float(input("Введите коэффициент b: "))
c = float(input("Введите коэффициент c: "))
if a != 0.0:
    x = (c - b) / a
    print(f"x = {x}")
elif b == c:
    print("x - любое число.")
else:
    print("Корней нет.")
```
